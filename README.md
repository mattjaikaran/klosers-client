# Klosers Front End

Front End MVP application for Klosers

## Technologies

- Node 18.13.0
- React 18
- Next 13.4
- TypeScript 5.1
- Redux Toolkit
- React Hook Form
- React Bootstrap
- SCSS

## Getting Started

```bash
$ git clone URL
$ cd REPO
$ yarn
$ yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
